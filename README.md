# Smart Home Care — Homepage + Caretaker View + Admin Dashboard (Vanilla)

## Pages
- `index.html` — Marketing homepage (animated + reviews slider + mini demo)
- `caregiver.html` — **Caretaker/Professional view** (job queue, OTP arrival verification, geo status, care notes + export)
- `dashboard.html` — **Admin dashboard** (KPIs, trend chart, risk signals, sessions table w/ filter + CSV export, verification queue)

## Run
Open any HTML file directly in a browser.

> Best experience (fonts + Swiper CDN): run a small server
- VS Code: Live Server
- Python: `python -m http.server 8080`

## Notes
- Index page uses Swiper via CDN for the Reviews slider.
- Caretaker + Dashboard are framework-free prototypes with simulated data.
